社内ローカルAI
メール校正ツール

ビジネスメールの誤字・変換ミス・表記揺れを検出し、専門用語を解説。社内のローカルLLMで動くため、データは社外に出さず・API課金もゼロ。利用はブラウザでURLを開くだけ。

OUR VALUE

提供できる価値

既存の外部校正サービスと違い、機密性の高い顧客メールも社内に閉じた環境で安全に校正できます。

機密性

入力したメールは社内サーバ内だけで処理。インターネットに送信しません。

ゼロ課金

即時性

導入容易

ARCHITECTURE

システム構成

「ブラウザ → Webアプリサーバ → ローカルLLM」の3層。すべて社内LAN内で完結します。

設計のポイント：利用者のメール本文は社内サーバ内だけで処理し、社外には一切送信しません。AIモデルはローカルGPUで動くため、機密データを第三者クラウドに渡さず、課金も発生しません。

FEATURES

主な機能

3種類の誤りを検出し、文中の専門用語には意味を添えます。

検出の種類内容例
スペルミス英単語・技術用語の綴り間違いLamda → Lambda
変換ミス漢字の同音異義語の誤り以外 ⇔ 意外、保証 ⇔ 保障
表記揺れ同一メール内の表記の混在サーバー ／ サーバ
専門用語の解説IT・ビジネス用語に意味を付与AWS, SLA, デプロイ …

※ 誤りは「赤字＋正しい語の注釈」、専門用語は「下線＋意味」で表示。本文の自動書き換えは行わず、指摘のみ（最終判断は人が実施）。

SCREEN

画面イメージ

左に本文を貼り付け、「分析開始」を押すと、右に注釈付きの結果が表示されます。
