# 功能说明（功能与场景）

> BrightStar 培训助手 —— 公司内部培训课程管理。学员在微信里聊天即可报名上课；
> 老师用文字命令管理课程；课程网站记成绩/问卷；可选 AI 问答。支持**日本語 / 中文**（默认日语）。

---

## 一、三种角色
| 角色 | 入口 | 能做什么 |
|---|---|---|
| 学员 | 微信客服号「BrightStar 研修アシスタント」 | 注册、看课、报名/取消、查下节课、拿登录码、AI 问答 |
| 老师 | 同一客服号（发文字命令） | 建课、发布、改课、删课、查学员、名单、随机分组 |
| 网站访客 | 课程网站（登录码登录） | 看讲义、做测验/课后问卷；老师可导出成绩 |

> 一人可**同时是老师和学员**：老师认证后仍能像学员一样报名。

## 二、学员功能
- **注册（两步）**：① 选语言（默认日语，回 1/2 或「日本語/中文」）② 回复真实姓名（2–20 字，姓名用于成绩与点名）。直接发姓名不选语言＝默认日语。
- **菜单/数字**：`1` 课程列表、`2` 我的课程、`3` 下节课（时间+Zoom）。
- **报名 / 取消**：`报名 课程名` / `取消 课程名`（日：`申込` / `キャンセル`）。退课截止＝开课前 2 小时。
- **下节课**：返回最近一节已报名课程的 JST 时间 + Zoom 链接。
- **网页登录码**：发「登录码」得 8 位码，**有效期 7 天**，过期再发即换新；用于登录课程网站。
- **改名**：`改名 新名字`（日：`名前変更`）。
- **切换语言**：发「语言/言語」，或直接「中文/日本語」。
- **🤖 AI 问答（可选）**：发「AI」开启 → 自由提问课程相关问题（由本地知识库 + Bedrock 作答）；发「退出AI」回普通模式。开启后具体操作（报名/数字）仍有效。

## 三、老师功能（文字命令）
先发 `老师认证 <口令>` 升级为老师（或被加入白名单）。命令：
| 命令 | 示例 |
|---|---|
| 建课（草稿） | `建课 Python入门\|2026-06-20 14:00\|90\|20\|零基础` |
| 发布（生成 Zoom 并对学员开放） | `发布 Python入门` |
| 改课 | `改课 Python入门\|容量=30;开课时间=2026-06-21 10:00` |
| 删课 | `删课 Python入门` |
| 学员列表 | `学员列表` |
| 报名名单 | `名单 Python入门` |
| 随机分组 | `分组 Python入门\|组数=4`（或 `每组=5`） |

> 时间一律 JST、格式 `YYYY-MM-DD HH:MM`。建课后**必须「发布」**学员才看得到。Zoom 当前为演示桩。

## 四、课程网站
- **登录**：输入微信发的登录码 → 进入（网站记住登录者**姓名**）。
- **学习**：看讲义（第1课 Amazon S3 等），右上可切日/中、深色、🎨配色；宽屏左侧大纲；动手操作附 AWS 官方图文教程链接。
- **测验 / 课后问卷**：提交后**按本人记录**（带姓名）。
- **成绩导出**：老师身份可按项目导出全部学员的成绩与问卷。

## 五、AI 问答（RAG）机制
- 仅学员**手动开「AI」**后才调用 Bedrock（控成本）。
- 流程：问题→Titan 向量化→在知识库（DynamoDB）暴力余弦检索 top-k→拼进 Claude（`jp.` 日本境内 profile，**数据不出日本**）→按学员语言作答。
- **资料外问题**会礼貌说明“没有相关资料”并引导发「菜单」，不乱编。
- 知识库内容可替换为你们真实的课程/FAQ（见使用说明的“灌知识库”）。

## 六、自动提醒
- 开课**前约 1 小时**，自动经微信客服号给已报名学员推送提醒（时间 + Zoom）。
- 限制：微信平台要求学员近 48 小时内与客服号有互动才能主动推送。

## 七、典型场景串讲
**学员**：进群 → 选「日本語」→ 填姓名「山田太郎」→ 发「AI」问「受講の流れは？」得到答复 → 回「1」看课程 → `申込 Python入门` → 开课前 1h 收到提醒 → 课后用「登录码」登录网站做问卷。

**老师**：发 `老师认证 BS-TEACHER-2026` → `建课 …` → `发布 …` → 开课前 `名单 Python入门` 查人 → `分组 Python入门|组数=4` 分组 → 课后网站导出成绩。

## 八、语言支持
- 学员级语言（注册时选、可随时切），机器人**所有回复**按个人语言；网站默认日语、可切中文。老师管理命令为中文。
