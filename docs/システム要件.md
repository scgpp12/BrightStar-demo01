# BrightStar 社内システム ｜ システム要件

## 1. 稼働環境（クラウド）
- **AWS アカウント**（東京リージョン ap-northeast-1）
- 利用サービス：Lambda / Lambda Function URL / DynamoDB / S3 / EventBridge / SSM Parameter Store /（任意）Bedrock
- すべて**従量課金・サーバーレス**。固定サーバ・常時稼働コストなし。

## 2. フロント（メッセージング）
- **LINE 公式アカウント（Messaging API チャネル）** 1 つ（人事専用を推奨）。
  - Channel secret / Channel access token（長期）
  - Webhook 利用 ON、応答メッセージ（自動応答・あいさつ）OFF
- 社員・人事は **LINE アプリ**（スマートフォン）。

## 3. 開発・デプロイ環境
- **Node.js 18+**（AWS CDK v2 / TypeScript）
- **AWS CLI**（認証情報設定済み、東京リージョン）
- Python ランタイムはクラウド側（Lambda Python 3.12 / arm64）。**ローカルに Python 依存のインストールは不要**（コードは標準ライブラリ＋boto3 のみ）。
- 初回のみ `cdk bootstrap`。

## 4. 機密情報の取り扱い
- LINE の Channel secret / access token は **SSM Parameter Store（SecureString）** に保存。
  - `/<app>/<stage>/line/secret`、`/<app>/<stage>/line/token`
- **git・メモリ・環境変数平文には保存しない**。CDK は参照名のみを扱う。

## 5. データ・保管ポリシー
- DynamoDB：On-Demand 課金、保存時暗号化、PITR 有効。
- S3：Block Public Access 全 ON、SSE、**バージョン管理 ON**。
- 提出物ライフサイクル：**60 日で Glacier Deep Archive（最安）へ移行、365 日で自動削除**（旧バージョン含む）。
  - ※ 2 か月以内は標準ストレージで即時ダウンロード可。2 か月超は復元（数時間）が必要。
- 一時データ：`pending/` 7 日、`exports/`(zip) 3 日で自動削除。

## 6. 運用上の前提・制約
- **社員名簿（roster）が主データ**。提出対象は名簿に登録された全社員（LINE 未登録でも一覧に表示）。
- LINE 登録は **氏名が名簿と一致**することが条件（部署は名簿から自動取得）。
- リマインドは LINE 連携済みの未提出者のみ配信可能。未連携者は一覧で `※Line未登録` 表示 → 人事が個別連絡。
- LINE Push はアカウントのメッセージ配信量に依存（未提出者のみ配信のため通常少量）。
- 同姓同名は名簿で一意に特定できないため、運用で回避（必要なら社員番号運用）。

## 7. コスト目安
- 数十〜数百名規模なら、DynamoDB/Lambda/S3 はいずれも**月数十円〜数百円程度**（従量）。
- 大きなコスト要因は基本なし（Bedrock を使わなければ AI 費用ゼロ）。

## 8. テンプレートの所定セル（内容チェック前提）
提出時の自動チェックは、様式の以下セルを参照します。**様式のセル位置を変更しないでください。**

| 種別 | セル | 内容 | チェック |
|---|---|---|---|
| 経費 | **A1** | 「YYYY年M月分経費」 | 年月が提出月と一致（不一致＝拒否） |
| 勤怠 | **B3** | 氏名 | 登録氏名と一致（不一致＝拒否） |
| 勤怠 | **B5〜** | 日付（1日〜月末、横並び） | 年月一致（拒否）＋日付の欠落（警告） |
| 勤怠 | **6・7行** | 時間・分（各日付列に対応） | 休日に時間入力があれば警告 |

- 年月・氏名の不一致は**保存しない**（修正のうえ再提出）。
- 休日勤務・日付欠落は**警告のみ**（保存はする）。
- 解析は標準ライブラリのみ。ふりがな（ルビ rPh）は自動除外、日付はシリアル値／文字列の両方に対応。

## 9. 拡張余地（任意）
- Excel 中身の解析（工数・金額集計）：openpyxl レイヤー追加で対応可。
- 人事による配信時刻のセルフ変更、複数事業所・多言語強化、他チャネル（Slack/Teams 等）展開。
