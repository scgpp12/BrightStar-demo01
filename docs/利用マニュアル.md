# BrightStar 社内システム ｜ 利用マニュアル（社員・人事向け）

LINE のトークだけで、毎月の **勤怠（作業時間記録簿）** と **通勤費（経費）** を提出・管理できます。

---

## はじめに（全員・初回のみ）
1. 人事から案内された **LINE 公式アカウント（人事アシスタント）** を友だち追加。
2. 何かメッセージを送ると、**お名前（フルネーム）** を聞かれます。
3. **社員名簿と一致する氏名**を送ると登録完了（部署は名簿から自動設定）。
   - 「名簿に見つかりません」と出たら、氏名の表記を確認するか人事へ連絡してください。

---

## 社員の使い方

### ① 様式（テンプレート）を取得
- `テンプレ` と送信 → 「作業時間記録簿」「経費」のダウンロードボタンが届きます。タップで Excel を保存。

### ② 提出する（再提出も可）
- 記入済みの Excel を **そのままトークに送信**。
- ファイル名から自動で種別を判定し、「✅ 2026-06 の作業時間記録簿を受け付けました」と返信。
- 種別が判別できない場合は「作業時間記録簿ですか？経費ですか？」と聞かれるので、どちらかを送信。
- 同じ月にもう一度送れば**上書き（再提出）**されます。

### ③ 自分の提出を確認
- `履歴` と送信 → これまでの提出一覧と、各ファイルの **ダウンロードボタン**が届きます。

> 💡 ファイル名の目安：勤怠は「作業時間記録簿（氏名）_YYYYMM.xlsx」、通勤費は「経費（氏名）_YYYYMM.xlsx」。様式のまま記入して送ればOK。

---

## 人事の使い方（人事ロールのみ）

### 提出状況を見る
- `一覧` … 当月の全社状況（**未提出が上・提出済が下**）。`一覧[202605]` で過去月も。
  - LINE 未登録の社員は `※Line未登録` 付きで表示されます。
- `未提出確認` … 当月の**未提出者だけ**を一覧（`未提出確認[202605]` で月指定／省略時は当月）。

### 督促する
- `催促` … 当月の未提出者へリマインドを LINE 配信（提出済は対象外）。`催促[202605]` で月指定。
  - 種別を絞る場合：`催促 勤怠` / `催促 経費`。
- 自動配信：毎月 **25・28 日 09:00（JST）** に未提出者へ自動リマインド。

### まとめてダウンロード
- `一括DL` … 当月の全提出を **1 つの zip** にまとめ、ダウンロードボタンを返信。

### 社員名簿の管理（増減・変更）
| 操作 | コマンド例 |
|---|---|
| 一覧 | `名簿` |
| 追加 | `社員追加 田中三郎 経理部`（人事にするなら末尾に `hr`） |
| 変更 | `社員変更 E003 部署 営業部`（項目：部署／名前／role） |
| 削除 | `社員削除 E003`（氏名でも可） |

> 🔔 名簿を追加・変更・削除すると、**人事メンバー全員へ変更通知**が自動配信されます。

---

## 提出が拒否されたとき（自動チェック）
- **「年月が一致しません」** … ファイルの年月（経費=A1／勤怠=B5）を**今月**に直してください。
- **「氏名が一致しません」**（勤怠） … 勤務表の **B3** を登録氏名（社員名簿どおり）に直してください。
- **「年月を確認できませんでした」** … 正しいテンプレートを使い、所定セルに年月を入力してください。
- **休日に勤務時間（注意のみ）** … 土日・祝日に時間が入っていると通知します。出勤が正しければそのまま、誤りなら修正して再提出してください。
- **日付の欠落（注意のみ・勤怠）** … 5 行目の日付が 1 日〜月末まで揃っていないと、未入力の日を通知します。日付を補って再提出してください。

## よくある質問
- **Q. 部署を間違えた／変わった** → 人事に `社員変更 <empId> 部署 <新部署>` を依頼。
- **Q. 提出したのに催促が来る** → 提出後は対象外になります。来る場合は別の月／別の種別が未提出の可能性。`履歴` で確認。
- **Q. スマホを変えた** → 新端末で友だち追加し、同じ氏名で登録すれば再連携されます。
- **Q. 2 か月以上前の古いファイルがダウンロードできない** → 低コスト保管（Deep Archive）へ移動済みのためです。必要時は人事へご相談ください。
