CASE STUDY ／ サーバーレス × LINE × CDK

BrightStar 社内システム

毎月の「勤怠（作業時間記録簿）」と「通勤費（経費）」の提出・督促・集計を、LINE のチャットだけで完結。社員名簿を主データに、未提出者を可視化し、自動リマインドと一括ダウンロードまで。AWS サーバーレス＋CDK(TypeScript) でフルスクラッチ開発。

OVERVIEW

概要

社員は普段使う LINE で、記入した Excel を送るだけ。人事は LINE で全社の提出状況を即確認し、未提出者へワンタップ督促、当月分を一括ダウンロード。社員名簿（DB）が「提出対象」を決めるため、LINE 未登録の社員も `※Line未登録` として可視化されます。固定費ほぼゼロの従量課金で運用できます。

FEATURES

主要機能

提出（再提出可）

勤怠・経費の Excel を送るだけ。ファイル名から自動分類し S3 保存＋受付記録。判別不能なら種別を質問。

様式ダウンロード

「テンプレ」で空白様式（勤怠・経費）をボタンから取得。長いURLはボタンに隠す設計。

個人履歴

自分の提出履歴とダウンロードボタンを表示。最新資料をすぐ確認。

未提出の可視化

「未提出確認[YYYYMM]」で未提出者を一覧。LINE 未登録者も `※Line未登録` 付きで表示。

自動・手動リマインド

毎月25・28日 09:00(JST) に未提出者へ自動配信。人事は「催促」で即時督促。提出済は対象外。

一括ダウンロード

「一括DL」で当月の全提出を 1 つの zip にまとめ、ボタンから取得。

社員名簿 CRUD（人事）

LINE から名簿の追加・変更・削除。変更は人事全員へ自動通知。

氏名だけで登録

初回は氏名を送るだけ。社員名簿と照合して本人特定、部署は名簿から自動取得。

締め後の再提出確認

月をまたいだ再提出は、人事全員へ確認通知を自動配信。

設計の勘所：社員名簿（DB）を主データにして「提出対象」を確定し、LINE 未登録者も可視化／資格情報は SSM SecureString で git に出さない／提出物DLは HMAC 署名付き短縮リンクで越権防止（LINE のボタンURL 1,000文字制限も回避）／保存は 2か月は即時DL、以降 Deep Archive で最安、1年で自動削除。Lambda は外部依存ゼロでデプロイが軽量。
